# Pantazi Plastic Surgery Website Package

This folder is ready to upload as a static website root.

Public root files:
- `index.html`
- service pages
- `assets/`
- `legal/`
- `cookie-consent.js`
- `assets/booking-modal.js`
- `robots.txt`
- `sitemap.xml`
- `site.webmanifest`

Online booking:
- Booking buttons open a Calendly modal using `https://calendly.com/glykeriapantazi/30min?hide_event_type_details=1&hide_gdpr_banner=1`.
- Calendly's external widget script is loaded only after the visitor clicks a booking button.

Internal handoff files:
- `SEO/SEO-map.md`
- `SEO/launch-checklist.md`
- `SEO/redirects.md`

Do not upload the `SEO/` folder if you do not want internal notes public.
